# srposition 设计概要

- pubdate: 2013-8-1

-------------------------------

## 概述

srposition是基于araleJS的地图标识插件

## 功能

1. [位置标识][#1]
   
   根据坐标点在图片上标识未知，利用tip显示位置信息

2. 事件绑定
   
   在标识的位置上绑定事件，提供操作
